Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web

How do you select an element from the DOM?

By document.get(ElementById,ElementsByClass)etc.

What is event Delegation in the context of the DOM?

Event Delegation is a pattern based upon the concept of Event Bubbling.

How do you manipulate an elements attributes and styles using the DOM?

By Using setAttribute() method
